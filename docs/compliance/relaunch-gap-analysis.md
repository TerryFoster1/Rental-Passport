# Relaunch Legal and Compliance Gap Analysis

Date: 2026-07-23  
Non-legal assessment: this document identifies legal-review needs; it does not declare compliance.

## Gaps Requiring External Legal Review

- PIPEDA consent, access, correction, retention, and deletion practices.
- CASL implications for employer, landlord, reference, and applicant emails.
- GDPR readiness if serving EU/UK residents.
- Credit authorization wording, retention, display, and permissible purpose.
- Employment verification consent and what may be asked.
- Reference questions by region.
- Previous landlord/property manager questions by region.
- ID document collection, retention, masking, and deletion.
- Protected characteristics and fair-housing restrictions.
- Public-record and tribunal searches.
- Regional lease requirements remain Rental District-owned.
- Escrow and financial-service claims are out of scope without legal review.

## Implementation Readiness

| Area | Readiness | Notes |
|---|---:|---|
| Consent records | Partial | Database and service inserts exist; final legal text and withdrawal effects needed. |
| Credit authorization | Partial | Fields exist; manual workflow and exact legal text needed. |
| Document retention | Missing | No retention/deletion automation proven. |
| Data export/deletion | Missing | No user-facing process implemented. |
| Protected characteristic controls | Planning | Must be built into forms, AI boundaries, reviewer training. |
| Public information checks | Missing | Boundaries documented separately; provider/legal approval required. |

## Launch Rule

If a claim touches credit, identity, employment, references, prior landlord comments, public records, fraud, or AI, it must be reviewed by counsel before public launch.
